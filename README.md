## Installation

``` bash
# Install Bun
curl -fsSL https://bun.sh/install | bash

# Clong this repository
git clone https://github.com/hangxingliu/ai-hub.git
cd ai-hub

# Install dependencies
bun install

# Start the server
bun start
```
